# 🧩 2048 Streamlit

A clean, minimalist clone of the classic 2048 sliding block puzzle game built natively in Python using [Streamlit](https://streamlit.io/).

## 🚀 Features
- **Clean UI**: Beautifully styled tiles with a glowing effect for your highest tile.
- **Keyboard Controls**: Seamlessly play with the keyboard arrow keys instantly (auto-injected JS for responsive inputs).
- **Responsive Layout**: Designed to be centered and look aesthetic on diverse screen sizes.
- **Minimal Dependencies**: The only external library needed is `streamlit`.

## 🛠️ How to Play
Combine blocks with the matching numbers to reach the ultimate **2048** tile!
* **Up / Down / Left / Right** moves the tiles respectively.
* When two tiles with the same number touch, they merge into one!

## 📥 Installation & Running Locally

1. **Clone the repository** (or download the script)
2. **Install Streamlit**
   ```bash
   pip install streamlit
   ```
3. **Run the Game**
   ```bash
   streamlit run 2048.py
   ```
